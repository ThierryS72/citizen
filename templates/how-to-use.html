Comment utiliser l'application Citizen

Création d'un compte utilisateur

Pour utiliser notre application citoyennes, vous devez juste créer un compte gratuit.

Vous pouvez ensuite avoir accès à toutes les fonctionalités de l'outil.

Déclaration d'un problème

Pour déclarer un problème, il vous suffit de cliquer sur la map pour y placer un marqueur (il est ensite possible de le déplacer pour ajuster sa position).

Lorsque le marqueur est positionné, cliquer sur celui-ci pour faire apparaître l'étiquette "Déclarer mon problème".

Cliquer ensuite sur le texte de l'étiquette pour accèder au formulaire de déclaration.

Formulaire de déclaration

Tous les champs du formulaire doivent être remplis pour que la déclaration soit validée (exceptions des mots clés facultatifs):

  • Description: description détaillée du problème
  • Image: URL de l'image correspondante au problème
  • Type: selection du type de problème dans la liste déroulante
  • Mots clés: insertion des mots clés qui permettrons la recherche efficace des problème (voir paragraphe ci-dessous: Page principale)

Cliquer ensuite sur le bouton "Déclarer un problème" pour valider le formulaire. Après celà le problème que vous avez décrit sera automatiquement ajouté à la liste des nouveaux problèmes.

Page principale

La page principale de Citizen app contient tous les problèmes ainsi que leurs situations sur la map.

Il est possible de filter ou des rechercher des problèmes en fonction du type de problème ou en fonction de mots clés (tag) définis lors de la déclaration.

Détails des problèmes

Depuis la Page principale ou depuis Mon compte, il est possible de cliquer sur un problème pour voir les détails de celui-ci.

Le détail des problèmes permet également l'ajout de commentaires précieux à son historique.

Des commentaires sur les actions sur les problèmes peuvent être ajoutés par le staff.

Administration

Le menu d'administartion permet de voir l'historique de vos actions ainsi que les actions en cours.

La liste des utilisateurs (seulement pour le staff), en dessous des actions en cours, permet de changer un utilisateurs de type citizen en staff.

Mon Compte

Le menu mon compte permet de mettre à jour vos données personelles et de voir tous les problèmes que vous avez déclarés.
